Tabla de Operadores en Programación

Operador Tipo Descripción
== Comparación Igualdad: compara si dos valores son iguales
!= Comparación Desigualdad: compara si dos valores son diferentes
> Comparación Mayor que: verifica si el primer valor es mayor al segundo
< Comparación Menor que: verifica si el primer valor es menor al segundo
>= Comparación Mayor o igual que: verifica si el primer valor es mayor o igual al segundo
<= Comparación Menor o igual que: verifica si el primer valor es menor o igual al segundo
AND (&&) Lógico Retorna verdadero si ambas condiciones son verdaderas
OR (||) Lógico Retorna verdadero si al menos una condición es verdadera
NOT (!) Lógico Invierte el valor booleano: verdadero se vuelve falso y viceversa
