# Tabla de Operadores en Programación

## Descripción

Este proyecto es una aplicación web interactiva que presenta una tabla completa de operadores fundamentales utilizados en programación. Incluye operadores de comparación y operadores lógicos, con descripciones claras de su funcionamiento.

La tabla está diseñada de manera responsiva y con estilos visuales que facilitan la comprensión y diferenciación entre tipos de operadores.

## Características

- Tabla interactiva con los operadores más comunes en programación
-  Diseño moderno y responsivo con gradientes y efectos hover
-  Distinción visual entre operadores de comparación (rojo) y operadores lógicos (verde)
- Compatible con dispositivos móviles y escritorio

## Operadores Incluidos

### Operadores de Comparación
- `==` - Igualdad
- `!=` - Desigualdad
- `>` - Mayor que
- `<` - Menor que
- `>=` - Mayor o igual que
- `<=` - Menor o igual que

### Operadores Lógicos
- `AND (&&)` - Conjunción lógica
- `OR (||)` - Disyunción lógica
- `NOT (!)` - Negación lógica

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica de la página
- **CSS3** - Estilos avanzados, gradientes y efectos de transición
- **Markdown** - Documentación del proyecto

## Estructura del Proyecto

```
mi-proyecto/
```

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador web
2. Alterna entre los operadores de comparación y lógicos para estudiar
3. Los estilos de color diferencian los tipos de operadores:
   - **Azul claro**: Operadores de comparación
   - **Verde claro**: Operadores lógicos

### Con Live Server (Recomendado)

Si tienes instalada la extensión Live Server en Visual Studio Code:
1. Click derecho en `index.html`
2. Selecciona "Open with Live Server"
3. La página se abrirá automáticamente en tu navegador
